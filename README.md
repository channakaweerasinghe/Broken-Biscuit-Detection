# Broken Biscuit Detection using Classical Image Processing

## Problem Description
This project detects biscuits in images and classifies them as intact or broken.

## Dataset
Images of round and square biscuits placed on white A4 paper.

## Tools and Libraries
- Python
- OpenCV
- NumPy
- Jupyter Notebook

## Methods Used
- Color detection (HSV)
- Morphological operations
- Contour detection
- Shape analysis

## How to Run
1. Open Anaconda Prompt
2. Activate environment:  conda activate ee4216
3. Run Jupyter Notebook
4. Open biscuit_detection.ipynb
5. Run all cells

## Output
Images are saved in output_images folder with labels:
- Intact Biscuit
- Broken Biscuit
